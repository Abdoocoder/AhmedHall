## البدء السريع

```bash
# التثبيت
npm install

# التشغيل
npm run dev
```

افتح http://localhost:3000

## التقويم النبطي

- كانون ثاني (يناير)
- شباط (فبراير)
- اذار (مارس)
- ... المزيد في الوثائق
